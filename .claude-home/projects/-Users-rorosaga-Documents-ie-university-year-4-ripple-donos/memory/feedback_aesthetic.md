The user shared a Studio Ghibli-style solarpunk illustration as aesthetic inspiration for the Donos frontend. Key visual qualities:
- **Vivid blue sky** with large, soft, painterly white clouds — not pale/washed out, actually vibrant
- **Lush, rich greens** — grass, vines, trees — nature is abundant and dominant
- **Technology softened by nature** — structures overgrown with plants, vines hanging from equipment
- **Warm sunlit feeling** — golden hour warmth, wildflowers in foreground
- **Hand-painted / illustrated quality** — not photorealistic, has an anime/watercolor brushstroke feel

**Why:** The user felt the first frontend attempt didn't nail the visual design. This reference makes the sky background more vibrant than the pale `#E8F0FA` currently in the frontend skill.

**How to apply:** When building the sky hero/celebratory sections, push toward a more vivid Ghibli blue (`~#7EC8E3` to `#4DA6D9` range) rather than the very pale `#E8F0FA`. Clouds should feel painterly and voluminous, not wispy CSS gradients. Greens should be rich and saturated. The overall vibe is "hopeful meadow at golden hour" not "sterile startup landing page".
